# 部署指南

## 第一步：将代码推送到 GitHub

1. **创建新仓库**
   - 访问 https://github.com/new
   - 仓库名称：`liushuiwu-forum`
   - 选择 Private（私有）或 Public（公开）
   - 不要勾选 "Add a README file"（我们已有）
   - 点击 "Create repository"

2. **本地初始化 Git 并推送**
   ```bash
   cd "C:\Users\huang\Desktop\流水无涯论坛-GitHub"
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/liushuiwu-forum.git
   git push -u origin main
   ```

## 第二步：启用 GitHub Pages

1. 进入仓库 **Settings**
2. 左侧菜单点击 **Pages**
3. 在 "Build and deployment" 部分：
   - Source 选择 **Deploy from a branch**
   - Branch 选择 **main**，文件夹选择 **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟，网站将在 `https://你的用户名.github.io/liushuiwu-forum/` 上线

## 第三步：将你的网站内容放入

将你本地 Vite 项目的 `dist` 文件夹内容复制到此目录，然后推送：

```bash
# 在你的项目目录执行构建
npm run build

# 复制 dist 内容到 GitHub 文件夹
# 然后推送
git add .
git commit -m "Deploy website"
git push
```

## 自动化部署（可选）

此项目已包含 GitHub Actions 工作流。当启用 GitHub Pages 后，每次推送到 main 分支会自动部署。

## 注意事项

- 如果网站无法访问，检查 Settings → Pages 中的配置
- 自定义域名可在 Pages 设置中添加
- 查看 Actions 页面可查看部署状态
